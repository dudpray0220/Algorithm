# 케이크 초 최소세트 구하기    
케이크 초 1세트에는 0부터 9모양이 있으며, 6과 9는 모양이 비슷하여 같은 취급으로 한다.    
손님들의 나이가 ages라는 리스트로 주어진다. 이때, 모든 손님 나이의 케이크초를 제공하기위해 준비해야할 초 세트의 최솟값은?
       
> 제한 사항    
> > ages의 길이 : 10,000 이하의 자연수    
> > ages의 원소 : 100이하의 자연수           
            
> 입출력 예
> > ages : {23, 61, 46} -> result : 1    
> > ages : {44, 33, 69, 62} -> result : 2
